# Phase 2 · Clarify

Discovery 에서 찾은 **UX 이슈 Top 5** 를 기반으로, **어느 이슈를 우선 해결할지** 확인.

Aesthetic 선택이 아님. 문제 해결 우선순위가 핵심.

## 질문 세트

### Q1. 우선 사용자 (어느 임상 역할?)
- 임플란트 전문의
- 일반 치과의 (진단 중심)
- 교정 전문의
- 범용

### Q2. 해결할 이슈 · multiSelect
Discovery 에서 찾은 Top 5 이슈를 옵션으로 제시. 사용자가 1-3개 골라서 "이것부터 고쳐줘" 표시.

예시:
- [ ] 숨겨진 탭 9개 복구 (TMJ/Ceph/Aligner 등)
- [ ] 워크플로우 단계 전환 클릭 감소
- [ ] OTF 패널 공간 효율
- [ ] 선택 객체 피드백 강화
- [ ] WW/WL 중복 접근 통합
- [ ] 키보드 단축키 체계

### Q3. 변경 허용 범위
| 옵션 | 의미 |
|------|------|
| 현재 탭 구조 유지 + 내부 최적화만 | 보수적 |
| 탭 그룹핑/재배치 | 중간 |
| 탭 체계 자체 재설계 (Phase 등 새 개념) | 공격적 |
| 완전 자유 · willa 재량 | 풀 자유도 |

### Q4. 디자인 톤 (보조 질문 · aesthetic 은 이슈 해결 후 결정)
간단히:
- 기존 UI 톤 유지
- 신선한 방향 환영 · willa 재량
- 프로토타입은 개념 시연용 · 디자인 중요도 낮음

## 규칙

- 한 번에 4개까지
- **Q2 가 가장 중요** — 해결할 이슈가 모든 제안의 기준
- 답 받으면 1줄 요약: "정리: 일반 치과의 · 워크플로우 전환 + 단축키 해결 · 탭 재배치까지 허용"

## 결정 로그

```
{
  targetUser: "일반 치과의",
  issuesToFix: ["workflow-transition", "shortcuts"],
  scope: "탭 그룹핑/재배치",
  tone: "willa 재량",
}
```
